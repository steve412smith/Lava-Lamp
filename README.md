# Lava Drift

A self-contained Android-friendly PWA for procedural lava-lamp / particle-field visuals.

## Open locally
Open `index.html` in a browser. Some offline install features require serving over HTTPS or localhost.

## Install on Android
1. Upload this folder to GitHub Pages, Netlify, Vercel, or any HTTPS static host.
2. Open the hosted URL in Chrome on Android.
3. Tap the three-dot menu.
4. Tap **Add to Home screen** or **Install app**.

## Controls
- Presets: Forest, Ocean, Ember, Aurora, Night
- Motion: animation speed
- Density: particle count
- Glow: bloom intensity
- Coherence: organized flow vs more turbulent flow
- Tap/drag on the animation to gently attract particles
